## What's working
- Mostly everything
- AppleID and iServices
- Wi-Fi & Bluetooth, Ethernet
- Display outputs DP
- Speaker, Headphone jack, microphone
- USB ports mapping
- Sleep/wake/shutdown
- Keyboard backlighting
- All Continuity & Handoff features (need to have Apple native WLAN card.

## What's not working
- SD Card reader
- Fingerprint Reader

## Not tested
- VGA output
- SIM card slot
- Smart card
- Docking station
